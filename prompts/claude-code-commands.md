* `/context` - summary of your current context
* `/simplify` - independent review of your code and architecture
* `/compact` - summarize and free up tokens from the context
* `/clear` - clear context
* `/rename` - name a session 
* `/color` - color a session
* `/hook` - immediate command to run
* `/agents` - create agents to run tasks
* `/loop` - run a job on schedule. to cancel use `/cronDelete`
* `/chrome` - allows browser to automate
* `/rewind` - previous step
* `/btw` - run a parallel question, without impacting current processing
* `/plan` - plan mode
* `/statusline` - print what you want to summarize at end of each prompt output
